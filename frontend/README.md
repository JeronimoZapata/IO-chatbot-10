# Frontend - ChatBot de Modelos de Simulacion

Frontend Next.js inicial listo para continuar en una etapa posterior.

## Requisitos

- Node.js 18+
- npm

## Instalacion

```bash
npm install
```

## Ejecutar en desarrollo

```bash
npm run dev
```

## Configuracion de entorno

1. Copia el archivo de ejemplo:

```bash
cp .env.example .env
```

2. Ajusta la variable si es necesario:

- `NEXT_PUBLIC_API_URL`: URL base del backend (por defecto `http://localhost:3001`).

## Nota

La integracion con el backend se realizara en una etapa posterior.
